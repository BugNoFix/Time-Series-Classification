README 

The project is organized in notebooks.

The first notebook is the one of DataExploration.ipynb (it does not modify 
the data).

The second notebook DataPrePrecessing.ipynb processes data. If changes on 
window and stride to build new pkl data to feed to the models are needed 
this is the to-go notebook.


The other notebooks are self-explanatory: they contain the models.

The Python files contain functions called by the models in the notebooks.

The library requirements are the standard ones for any Deep Learning 
environment.
